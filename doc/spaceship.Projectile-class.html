Base class for all projectiles. Provides mechanics for projectile animation. Subclasses that need to provide own mechanism should override methods _initialize_image and _update_image.

_initialize_image(self)

source code 
Initialize projectile's image and blit first state. Initialize frame_count and frame_length. This method may be overriden by subclasses. By default state 'frame0' of resource 'projectile' is blit.

_update_image(self)

source code 
Update projectile looks based on anything you wish. This method may be overriden by subclasses. By default subsequent stages from 'frame0' to 'frameN' of resource named base_res_name are cycled (N is equal to frame_count.

update(self)

source code 

Update projectile's image, position, detect collisions and dispose of the projectile if it leaves the screen.

This method may be overriden provided it does all mentioned above.
