_setattrs(self, params, values)

source code 

Setup instance attributes.

This is an auxiliary method that will set value of instance attribute if its name exists in params and values (as a key). All attributes whose names exist in params should have default values. If there is no default value for attribute which does not have value defined in values an exception is raised.
Parameters:
  • params (string, list or tuple) - Names of allowed parameters - if string, names should be comma-separated
  • values (dict) - Dictionary containing instance attribute values
