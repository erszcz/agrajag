BasicAutoShield, EnemyShipShield, SuperShield
Base class for all automatically activated shields. AutoShield is activated automatically when projectile moving with speed lesser than critical_speed approaches owner so that it could hit it in next iteration. Automatically activated shield is also automatically deactivated. If player activates shield manually it cannot be automatically deactivated unless there is no more energy.
