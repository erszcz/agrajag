Class Overlay

source code

          object --+    
                   |    
pygame.sprite.Sprite --+
                       |
                      Overlay
Overlay can be used to display auxiliary animated effects over a sprite. Overlays are not independent game objects. Overlays do not contain any graphics resources on their own. Overlays do not collide. Overlay's image size is changed dynamically. Overlay's image has to be initialized with owner's image in order to preserve surface properites. Overlay itself should be added to group 'draw' immediatelly after its owner is added to that groups in order to display properly (not to display over other objects).
