Base class for all shield types used both by player ship and enemies. A working shield may and should be represented by some graphics, but should not collide on its own.

Activate or deactivate the shield. Display shield graphics if shield is activated.

absorb(self, damage, efficiency)

source code 
If shield is active absorb specified amount of raw damage caused with efficiency and return remaining raw damage to be absorbed. Turn shield off it current energy drops to zero. If shield is not active, return unmodified raw damage.

recharge(self, supply)

source code 
Recharge shield energy. Amount of energy recharged depends on amount of energy supplied and recharge rate. It is assumed that supply is never larger than demand. If there is any excess it is lost.
