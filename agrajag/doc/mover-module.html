Module mover

source code

Module contain definitions of different types of movers (objects thay control movement of other objects).

All non abstract movers must have the same constructor signature. This allows to dynamically create movers for objects generated according to stage schemas.

Mover constructor takes exactly three parameters. First one is object's current position, second one is object's maximal speed and third one is a dictionary containing parameters specific for different classes.
