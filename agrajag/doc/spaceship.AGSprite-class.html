_init_animation(self, res, period, pos=(0, 0), align='center')

source code 
Initialize animation of resource res on object's overlay. If object's overlay is not initialized, do it.
Parameters:
  • res (string) - Name of graphic resource contained by self.gfx
  • period (float) - Animation time in seconds.
  • pos (sequence) - Position of a point relative to which image drawn is aligned. Point cooridinates itself must be relative to self.center.
  • align (string or sequence) - String or sequence used to align drawn images relative to pos (as when aligning AGRect.

_initialize_position(self, pos, align, size)

source code 
Initializes object's rect attribute which is required by pygame to render object's image. Also sets object's pos and align attributes.
Parameters:
  • pos (tuple or list of two integers) - position of arbitrary object's point
  • align (tuple or list of two strings) - names of properties used to align object's rect
  • size (tuple) - object's size in pixels

_update_position(self)

source code 
Set new position of the object. New position is determined by object's mover if object has one. If not position is not changed. Object's align attribute is used to properly align object's rect.
